# Security Policy

Report privately:

- credential exposure;
- unintended screenshot or log upload;
- arbitrary code execution;
- unsafe task files;
- dependency compromise;
- privilege escalation;
- failure of premium-currency protection;
- emergency-stop bypass.

Do not submit code that:

- reads or writes game memory;
- injects DLLs or loads drivers;
- bypasses or evades anti-cheat;
- captures credentials;
- accesses private game protocols;
- hides the automation from detection;
- enables multi-account cloud control.
