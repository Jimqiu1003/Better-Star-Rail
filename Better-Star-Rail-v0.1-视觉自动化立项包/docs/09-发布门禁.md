# 09 发布门禁

## 功能

- [ ] 自动启动通过。
- [ ] 开拓力识别和资源保护通过。
- [ ] 至少一个副本完整循环通过。
- [ ] 委托领取与重新派遣通过。
- [ ] 邮件、每日训练和已适配活动奖励通过。
- [ ] OCR/模板动作有动作后验证。
- [ ] 录制只捕获目标窗口内的用户动作。
- [ ] 无人值守具备运行上限和看门狗。
- [ ] 全局停止在所有任务中有效。

## 红线审查

- [ ] 无内存读写。
- [ ] 无 DLL 注入、Hook 或驱动。
- [ ] 无反作弊绕过或规避。
- [ ] 无私有协议和未公开 API。
- [ ] 无账号凭据。
- [ ] 无星琼消费。
- [ ] 无任意脚本/插件执行。
- [ ] 无多账号或远程云控。

## 隐私与安全

- [ ] 截图和日志只保存在本地。
- [ ] 导出前预览并提醒脱敏。
- [ ] 没有静默遥测。
- [ ] 没有隐藏后台进程。
- [ ] 没有静默开机启动。
- [ ] Release 哈希已发布。
- [ ] 依赖和许可证清单完整。

## 文案

- [ ] 首屏注明非官方。
- [ ] 明确可能违反条款并导致处罚。
- [ ] 不承诺安全、不封号或未来兼容。
- [ ] 不要求用户关闭反作弊。
- [ ] 不把安全软件警报一概称为误报。

## 发布批准

```text
Product: APPROVED
Technical: APPROVED
Compliance/Risk: APPROVED WITH KNOWN HIGH RISK
```

三项都存在书面记录后才能发布。
