# 05 绝对禁止项

## 反作弊与规避

禁止开发、接受贡献或讨论实现：

- 关闭、卸载、绕过或欺骗反作弊。
- 检测反作弊扫描时自动隐藏。
- 进程名、窗口名、签名或文件哈希伪装。
- 随机化轨迹、时间和行为以模拟真人或降低检测概率。
- 内核驱动、句柄提权、调试器规避。
- 远程下发反检测策略。

## 游戏进程和协议

禁止：

- ReadProcessMemory / WriteProcessMemory。
- DLL 注入、Remote Thread、图形 Hook。
- 扫描模块、偏移、实体或战斗数据。
- 抓包、解密或复现私有协议。
- 调用未公开 API。
- 修改客户端、配置、资源包或游戏文件。

## 账号与支付

禁止：

- 采集或存储账号密码。
- Cookie、Token、验证码和浏览器会话。
- 自动充值、购买付费商品。
- 自动消耗星琼。
- 账号托管、租赁、交易和代练。
- 多账号农场和云端批量控制。

## 高损失操作

v0.1 禁止：

- 遗器分解、销毁、强化。
- 角色、光锥和遗器养成。
- 抽卡。
- 商店购买。
- 好友删除、社交消息发送。
- 邮件删除。
- 修改账号或隐私设置。

## 发布和宣传

禁止：

- 宣称不会封号或绝对安全。
- 冒充官方或使用官方视觉身份。
- 引导用户关闭反作弊。
- 把安全软件警报统一描述为误报。
- 隐藏真实功能或使用无害名称规避审核。
