# Better Star Rail v0.1 — A 方案立项包

> 项目定位：面向 Windows 的《崩坏：星穹铁道》非官方视觉自动化工具。  
> 风险等级：高。第三方软件、模拟操作与无人值守功能可能违反游戏用户条款，并可能导致账号处罚、功能失效、软件被拦截或项目被要求停止。

## 1. 核心决策

v0.1 采用 **A 方案：屏幕视觉识别 + OCR + 前台键鼠模拟 + 有限状态机**。

项目明确接受类似 BetterGI 的技术风险，但不承诺：

- 不封号；
- 永久可用；
- 不被反作弊、安全软件或系统拦截；
- 不发生误操作；
- 与未来游戏版本兼容；
- 权利方认可或允许本项目。

免责声明不能免除开发者依法承担的责任，也不能改变游戏运营方依据用户条款采取措施的权利。

## 2. v0.1 必须完成

- 自动启动官方游戏或启动器。
- 自动消耗开拓力。
- 自动进入指定副本并使用游戏内自动战斗。
- 自动循环挑战、结算和领取奖励。
- 自动完成或领取委托。
- 自动领取邮件、每日训练与活动页面中的可领取奖励。
- OCR/模板识别后自动点击。
- 键鼠录制、回放和任务编排。
- 在用户明确启用后进行有限的无人值守运行。
- 运行日志、截图证据、异常恢复与紧急停止。

## 3. 绝对红线

即使用户主动要求，也不进入本项目：

- 读写游戏内存。
- DLL 注入、图形 Hook、内核驱动。
- 禁用、绕过、欺骗或探测反作弊。
- 隐藏进程、伪造签名、随机化行为以规避检测。
- 抓包复现私有协议或调用未公开接口。
- 收集账号、密码、Cookie、Token 或验证码。
- 自动充值、购买付费商品或消耗星琼。
- 多账号托管、代练、账号交易和远程云控。
- 修改或重新分发官方游戏文件。
- 宣称“安全”“零封号”“官方认可”。

## 4. 开源与代码来源

- 项目建议采用 `GPL-3.0-only`。
- BetterGI 可以作为架构和交互参考；复制其 GPL 代码时必须保留版权与许可证，并履行 GPLv3 的源码分发义务。
- 没有明确许可证的仓库不得复制代码、素材、OCR 模板、配置或文案。
- 官方游戏素材不得直接打包进发布文件；识别模板优先通过用户本机截图生成，或使用团队原创/明确授权的测试素材。
