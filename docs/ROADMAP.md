# Roadmap

V0、V1、V2 表示开发阶段；M0、M1、M2 表示里程碑。当前开发版本是 `0.2.0-dev`，`v0.1.0` 是未来产品发布版本。

## V0 — Completed

- 仓库治理、合规边界与 Apache-2.0 clean-room 基线。
- C# 14、.NET 10 LTS、WPF、MVVM、win-x64 工程骨架。
- 本地配置、日志、测试、Build、CodeQL 与依赖治理。

## V1 — Completed on feature branch, pending integration

- 项目自建测试窗口。
- 窗口、客户区、DPI、截图、识别、安全守卫、定向输入、动作后验证与急停闭环。
- 不包含真实游戏业务、无人值守、宏录制或自动刷本。

## V2 — Planned

V2 尚未开始。其范围、里程碑和安全门禁必须由项目负责人另行确认后才能实施。
